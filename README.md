# is-strictly-seven

![npm version](https://img.shields.io/npm/v/is-strictly-seven)
![npm downloads](https://img.shields.io/npm/dt/is-strictly-seven)
![License](https://img.shields.io/npm/l/is-strictly-seven)

A lightweight, zero-dependency TypeScript utility to check if a number is strictly equal to 7.

## Installation

Using npm:

```bash
npm install is-strictly-seven
```
